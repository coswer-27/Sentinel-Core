# 🛡️ Sentinel-Core

AI 驅動的即時語意分析安全插件。

## 🚀 快速開始 (Installation)

### 1. 後端 AI 引擎 (Core-Engine)
1. 進入目錄：`cd core-engine`
2. 安裝套件：`pip install -r requirements.txt`
3. 啟動服務：`python main.py`
   *(首次啟動會自動下載 BERT 模型，約 700MB)*

### 2. 前端瀏覽器插件 (Browser-Ext)
1. 開啟 Chrome 瀏覽器，進入 `chrome://extensions/`
2. 開啟右側「開發者模式」
3. 點擊「載入解壓縮後擴充功能」，選擇 `browser-ext` 資料夾
